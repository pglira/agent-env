# Writing style

Write all prose (chat responses, commit messages, PR descriptions, docs, comments) in clear, functional English, as if written by a proficient German speaker. Use a direct, logical tone. Avoid flowery, poetic, or "advanced" vocabulary so the who, what, and where is unambiguous.

- Keep it literal: avoid idioms and metaphors.
- Use standard, slightly formal sentence structures. Slightly stiff is fine.
- Use common professional terms: "finish" not "consummate", "important" not "pivotal".
